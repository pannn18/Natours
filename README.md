# 🏞️ Natours

Natours adalah website bertema petualangan yang dirancang dengan tampilan visual yang menarik dan pengalaman pengguna yang halus.  
Proyek ini dibangun menggunakan **HTML** dan **SCSS* serta menonjolkan desain yang bersih dengan animasi yang lembut dan layout responsif.

---

## ✨ Fitur Utama
- 🌱 Desain modern yang terinspirasi dari tema alam  
- 🎬 Animasi CSS yang halus untuk meningkatkan pengalaman pengguna  
- 📱 Tampil optimal di perangkat desktop, tablet, hingga smartphone  
- 🧭 Kartu tur interaktif yang menampilkan berbagai paket petualangan  
- 🗂️ SCSS modular dengan metode BEM untuk struktur kode yang rapi  

---

## 🛠️ Teknologi yang Digunakan
| Teknologi | Keterangan |
|----------|------------|
| **HTML5** | Struktur halaman dengan markup semantik |
| **SCSS / CSS3** | Layout, styling, animasi, dan komponen |
| **Vercel** | Hosting & deployment |
